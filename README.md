# Companies List
Тестовое задание для Лайфхак Студио

Приложение со скписком компаний и детальной информации о них.

### Стек:
 - Kotlin
 - MVVM
 - Hilt
 - Jetpack Compose
 - Material You(3)
 - Jetpack Navigation
 - OkHttp3, Retrofit, Coil
 - Coroutines
